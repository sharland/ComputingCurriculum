About this template

This template is based on Grayscale which is a premium quality, free Bootstrap 3 theme created by Start Bootstrap. You can use this template for any purpose, personal or commercial.

This striking, black and white theme features stock photographs by Gratisography along with a custom Google Map skin courtesy of Snazzy Maps.

With this template, just the slightest splash of color can make a huge impact on the overall presentation and design.

Use of Sections

As you can see this template works on the basis of sections which can have different styles including images applied to them. Take a look at the source code for the page and see if you can identify each section and what each tags do.

An example button

Customising sections

Each section is easily customisable and it is also quite easy to copy a simple section structure and see it appear on the page. Each section is defined with <section></section> tags and the opening section tag will have a unique id attribute giving a name for the section. The id attribute will be used as a link for the menu.

Use <div></div> tags inside to create the structure for the section. Each <div> will have a class which defines styles for that div.
